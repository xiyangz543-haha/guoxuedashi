# 场景模板

> 常见咨询场景的回答框架。


---


## 场景一: 概念溯源


```
用户问: 「[概念]」是什么意思？出自哪部典籍？

回答结构:
1. 概念的核心定义（引用 concept_glossary.md）
2. 最早或最重要的出处（引用具体典籍）
3. 历代演变（不同时期的不同理解）
4. 相关概念辨析
```

## 场景二: 立场对比


```
用户问: 儒家和道家对[某问题]的看法有什么不同？

回答结构:
1. 儒家的基本立场及典籍依据
2. 道家的基本立场及典籍依据
3. 核心分歧点
4. 可能的内在联系或互补
```

## 场景三: 现实映射


```
用户问: 古人如何看待[某现象]？对今天有什么启示？

回答结构:
1. 典籍中相关记载综述
2. 不同时代的观点演变
3. 古代智慧与现代语境的适配性分析
4. 需注意的时代差异和适用边界
```

## 场景四: 文献考证


```
用户问: [某句话/某段文字]出自哪部典籍？版本情况如何？

回答结构:
1. 出处确认（具体典籍、篇目）
2. 版本信息（如已知）
3. 异文情况（不同版本的差异）
4. 历代注疏家的重要解读
```

## 场景五: 跨库专题


```
用户问: 我想研究[某专题]，该从哪些典籍入手？

回答结构:
1. 专题所属部类
2. 核心典籍推荐（优先级排序）
3. 辅助参考（相关部类中的延伸材料）
4. 研读顺序建议
```
